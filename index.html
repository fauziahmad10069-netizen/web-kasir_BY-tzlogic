Integrasi & Pengaturan Google Sheets

Hubungkan aplikasi POS Web Anda dengan Google Sheets untuk sinkronisasi inventaris dan penjualan secara otomatis.

Cara Menghubungkan ke Google Sheets Anda

1

Buat Spreadsheet

Buat Google Sheet baru dengan sheet bernama Barang, Penjualan, dan Piutang.

2

Pasang Google Apps Script

Gunakan fitur Extensions > Apps Script pada Google Sheets untuk membuat API web-app endpoint yang dapat menerima request GET/POST dari aplikasi POS ini.

3

Hubungkan Web POS

Salin URL Web App yang dideploy dari Google Apps Script ke form di bawah, lalu aktifkan mode sinkronisasi otomatis.

Konfigurasi API Google Sheets

Salin endpoint URL yang didapatkan setelah mendeploy Apps Script sebagai "Web App" di Google Sheet Anda.

Transaksi baru akan langsung disinkronisasi ke Google Sheets sesaat setelah transaksi selesai.
